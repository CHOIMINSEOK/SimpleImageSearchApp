# SimpleImageSearchApp
### Features
1. 이미지 검색 & 리스트 탭
- 키워드 검색으로 이미지 리스트 불러오기
- 즐겨찾기 
- 선택시 상세화면으로 전환

2. 즐겨찾기 탭
- 리스트 탭에서 즐겨찾기 한 이미지 리스트

3. 리스트 탭과 즐겨찾기 탭 동기화 


### Stack
- Language: Kotlin
- AAC: ViewModel, LiveData, Room
- Network: Retrofit2, OkHttp3, Gson
- Async Handling: RxJava2
- DI: Koin  
- Image Loader: Glide
